# For-webpages
for creating reference links for files in webpages. this repo contains src files.

##### Example
```html
<html>
<body>
<img src="https://raw.githubusercontent.com/nkpro2000/For-webpages/master/nk.png" witdh=26 height=31>
</body>
</html>
```
